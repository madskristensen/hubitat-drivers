## Governance Principle

OAuth callback pattern is architectural debt-avoidance only when:
1. Vendor has a public OAuth Authorization Server with public client registration
2. Hubitat's callback URI can be whitelisted at vendor
3. Pattern reduces user friction vs. current UX

When vendor provides no public portal (Gemstone, SunStat), pattern is not applicable.
