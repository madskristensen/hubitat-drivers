## Key Design Decisions Applied

- **Cognito SRP auth:** `USER_SRP_AUTH` flow with Groovy HMAC-SHA256 math; refresh tokens stored in device state for token rotation
- **First-device binding:** Deterministic discovery (walk home groups in order, bind first device found). Logged for transparency.
- **Optimistic updates:** State changes immediately; reconciled via 30s polling against `/deviceControl/currentlyPlaying`
- **Capability mapping:** Hubitat standard color maps → Gemstone ARGB int encoding; brightness 0–100 → 0–255
- **Polling schedule:** `runEvery30Minutes()` initialized at driver install and hub startup
- **Error handling:** Graceful recovery on Cognito timeout, HTTP 401 (token refresh), 5xx gateway errors
- **Logging:** `logEnable` preference (auto-disabled 30 min) + INFO level for state changes + WARN for HTTP errors
