## Lesson Learned

Hubitat sandbox enforcement happens in two layers:
1. **Parse-time:** Static field initializer validator rejects ANY cross-`@Field` reference (GString or concatenation)
2. **Runtime:** JDK API blocklist (System.*, Thread.*, Runtime.*, reflection, Date, File, Eval, UUID, etc.)

Future Hubitat drivers must pass both layers. Initial testing should include actual Hubitat IDE parse + a sandbox audit sweep before shipping.
